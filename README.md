평소 애용하는 음성커뮤니케이션 소프트웨어인 디스코드 내에서 작동가능한 봇

## 🧑‍🤝‍🧑 멤버구성
임호정

## ⚙️ 개발 환경
- Java 11
- JDK 11.0.15
- IntelliJ

## 📌 주요 기능
1. 사용자 태그 시 사용자 프로필 사진 확대 기능
   - 디스코드에는 사용자 프로필을 한정된 크기로만 볼 수 있는 불편함이 있어 해당 기능을 만듬
2. 친구의 당직, 오프 알림
   - 서버에 !은비, !은비 내일, !은비 모레 입력 시 당직과 오프를 알려주는 기능
